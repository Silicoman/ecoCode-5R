Avoid using arrays in foreach loops

foreach deduplicates items in a list before starting the enumeration. It is therefore generally more economical to use a simple for loop when you have a good command of the collection.

Noncompliant Code Example

  $a = [1, 2, 3, 17];
    function testForEachLoop() {
        $dummy = 0;
        foreach ($a as $v) { // NOK
            $dummy += $v;
        }
    }

Compliant Solution

  $a = [1, 2, 3, 17];
  function testForLoop() {
        $dummy = 0;
        for ($i=0; $i < sizeof($a); ++$i) {
            $dummy += $a[$i];
        }
    }
